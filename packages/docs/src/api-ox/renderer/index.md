# renderer

The entry point for Gunshi renderer.

## Example

```js
import { renderHeader, renderUsage, renderValidationErrors } from 'gunshi/renderer'
```

**[Source](https://github.com/kazupon/gunshi/blob/main/renderer)**

## Functions

| Function                                                                       | Description                   |
| ------------------------------------------------------------------------------ | ----------------------------- |
| [renderHeader](/api-ox/renderer/functions/renderHeader.md)                     | Render the header.            |
| [renderUsage](/api-ox/renderer/functions/renderUsage.md)                       | Render the usage.             |
| [renderValidationErrors](/api-ox/renderer/functions/renderValidationErrors.md) | Render the validation errors. |
