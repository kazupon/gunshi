# my-tool(1) -- A utility for processing data

## SYNOPSIS

my-tool <OPTIONS>

## DESCRIPTION

A utility for processing data

## OPTIONS

- `-h`, `--help`
  Display this help message

- `-v`, `--version`
  Display this version

- `-i`, `--input` <input>
  Input file path

- `-o`, `--output` <output>
  Output file path (defaults to stdout)

- `-f`, `--format` <format>
  Output format (json, yaml, xml)

- `-V`, `--verbose`
  Enable verbose output
